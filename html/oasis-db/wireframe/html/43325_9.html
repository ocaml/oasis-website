Upload your own package

To create a package, you need to add a "_oasis" file in the toplevel directory of your project and upload a tarball. You can also advertise for the Bocage project on your project webpages.
We really appreciate to get your feedback, through bug reports or features requests.

Create a watch file to automatically upload your package

You can create a watch file to take care of automatically upload new versions to OASIS DB.

Get involved into development

Goto http://oasis.forge.ocamlcore.org for more information about the development of OASIS DB.
